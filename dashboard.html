Apps Script를 웹 앱으로 배포한 후 생성된 URL을 입력하세요
설정 방법
  1. 앳홈 공동구매 총괄 시트 열기 → 확장 프로그램 → Apps Script
  2. 프로젝트의 apps-script.js 내용 전체 붙여넣기
  3. 필요 시 파일 상단 CONFIG 섹션에서 시트·컬럼명 재확인
  4. 배포 → 새 배포 → 웹 앱 선택
  5. 액세스: 모든 사용자(익명 포함) 설정 후 배포
  6. 생성된 URL을 위 입력란에 붙여넣고 저장
